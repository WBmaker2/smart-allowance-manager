# 우리 반 꼬마 CEO: 스마트 용돈 기입장

5~6학년 학생이 용돈 사용 내역을 기록하고 원그래프로 소비 습관을 돌아보는 금융 교육 웹앱입니다.

- 배포 URL: https://wbmaker2.github.io/smart-allowance-manager/
- GitHub 저장소: https://github.com/WBmaker2/smart-allowance-manager

## 수업 맥락

- 대상: 초등학교 5~6학년
- 과목: 실과 / 수학 융합
- 성취기준: [6실02-02], [6수04-03]

## 수업 흐름 예시

1. 용돈 사용 경험을 나누며 수입, 지출, 잔액의 의미를 확인합니다.
2. 가상 예산이나 실제 용돈 상황을 정해 소비 기록을 입력합니다.
3. 항목별 지출을 원그래프로 확인하며 가장 큰 소비 비중을 찾습니다.
4. 필요한 소비와 줄일 수 있는 소비를 구분해 봅니다.
5. 다음 주 소비 계획이나 절약 목표를 짧게 정리합니다.

## 교사 안내

- 입력한 기록은 브라우저의 localStorage에 저장됩니다.
- 서버 저장, 계정 생성, 로그인 기능은 없습니다.
- 공용 기기를 사용할 때는 활동 후 전체 기록 지우기를 안내해 주세요.
- 실제 용돈 공개가 부담스러운 경우 가상 예산 활동으로 운영할 수 있습니다.
